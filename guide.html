划好领地范围-输入圈地指令

res领地教程-为自己的家设置保护

首先圈地工具为木锄头

输入“/res ?”查看全部sub命令简要说明；

输入“/res info”可以查看当前领地详细信息；

输入“/res list”列出自己所拥有的领地；

输入“/res listall”列出所有领地。

一、设置领地：

先用一个木锄头左键敲击一方块设置点A，右键敲击一方块设置点B（输入“/res select size”查看所选区域的大小）；

输入/res select size 即可查看你所选择领地的大小和花费。如果游戏币不够是不能创建领地的之后输入“/res create 123”

（例123为你领地的名字，更换成你想要设置的，区分大小写且不能与他人重复）这样设置后，就形成了[以AB连线为体对角线的长方体或正方体（包括A、B所在边）

二、移除领地：

输入“/res remove 123”（例）移除名为123的领地，注意，移除领地无法从服务器获得金钱！

输入“/res set 123 flags true/false”（例）来设置权限，其中：

①123是领地名，不填则默认为当前所处领地；②flags代表权限

权限列表：

权限名称 权限解释

move 移动 build 改变地形 place 放置物品 destroy 破坏 use 使用 container 打开箱子 pvp 玩家伤害 tp 传送到该领地 ignite 点火 firespread 火的传播 bucket 桶的使用 lavaflow 岩浆流动 waterflow 水的流动 flow 岩浆流动和水的流动 creeper JJ怪爆炸 tnt Tnt爆炸 monsters 怪物出生 animals 动物出生 subzone 子空间 healin 治疗

③true/false是权限状态，若输入true则为开启，false则为关闭。

四、子空间设置：

如同设置领地一样选择区域，不同的是之后输入“/res subzone 123 abc”（例）设置母领地为123的名为abc的子空间，该区域的权限设置和移除则需要将领地名变为123.abc，

无则默认为当前所处领地（在子空间则为子空间）。

五、领地信息设置：

可以设置进出领地时的提示信息，输入“/res message 123 enter(leave) ABC”（例）设置领地123的进入（离开）时的提示信息为ABC，

123处不填，则默认为当前所处领地；

特别地，在提示信息中可以添加：

%player - 在提示信息中显示进出领地的玩家名

%owner - 在提示信息中显示领主

%residence - 在提示信息中显示该领地

六、玩家指令：

/res default [领地名] 重置某领地的权限设置

/res give [领地名] [玩家名] 将领地赠与某玩家，你必须是领主且被赠予玩家在线

/res lists 预定领地许可清单的详细信息

/res message [领地名] [enter/leave] [信息] 设置进入/退出某领地时候显示的信息

/res message [领地名] remove [enter/leave] 移除进入/退出某领地时候的信息

/res mirror [源领地名] [目标领地名] 复制源领地的权限设置至目标领地

/res rename - [旧名称] [新名称] 重命名领地.对于子空间旧名称必须全名，新名称可以只写子空间名

/res renamearea [领地名] [旧名称] [新名称] 重命名某领地中某区域的名称

/res tp [领地名] 传送至某领地

/res tpset 设置当前领地的传送点为你站立的地方

/res unstuck 将你从当前领地移出

如何快速赚钱/发家

1.把自己挖到的物品出售给收购商人(主城NPC)这样可以赚取一笔不菲的初始资金，但是收购商人只收购一部分物品

2.物品道具还可以上架市场(/market create <手上物品价格> <出售多少个数量> 来把手上的物品道具上架到市场里面让其他玩家来购买)

3.有了初始资金和圈地然后开一个商店!普通玩家可以创建三十个商店摊位哦，开商店指令/qs

放置一个箱子手上拿着需要出售的物品输入价格创建完成就完成了

武器淬炼玩法说明

 

装备淬炼系统：

1.签到、在线、杀怪、金子商城可以获得淬炼宝石

2.淬炼宝石分为普通、中等、高等、完美几个等级

3.品质越高的宝石，越适合高星级的装备进行淬炼

◆ 1-2星武器用普通级宝石或者完美级宝石进行淬炼没什么区别

◆ 12-13星级用普通级宝石与完美级宝石进行淬炼就是天差地别

4.淬炼宝石可以淬炼武器和护甲

5.淬炼成功装备星级提高1星，淬炼星级最高18星

6.淬炼失败装备星级将会降低0-3星

7.全身装备达到套装星级附加属性和特效

◆ 举个栗子：⑥星级套装，⑨星级套装，⑫星套，⑮ 星套 ，⑱星套

8.金子商城有淬炼保护符出售，可保护装备3，6，9，12星装备的升级

◆ 受淬炼保护的装备失败不会低于所保护星级

◆ 但是要求装备星级大于等于所保护星级

 

每个星套都有专属的粒子特效和属性加成

淬炼作用：大幅提高自身实力

特色功能玩法说明

 

QS快捷商店

拿着要卖的物品 左键点击箱子 会提示你输入价格 就OK了,对了要记得把你卖的东西放入箱子里. 东西卖完会提示卖完信息给你的.

/qs buy 修改商店为收购

/qs sell 修改商店为出售

/qs price 修改商品价格更

更多资料请输入指令/mc

 

全球市场

在主城找npc进入市场，或输入指令/market listings

购买后输入 /market mail 来取出购买物品

出售物品需要将物品拿在手上然后输入/market create 总价 数量

更多资料请输入指令/mc

 

地皮系统

/plotme claim 为别人认领你脚下的地皮

/plotme auto 自动认领顺序下一个地皮

/plotme home[:#] 设置家，如果你有多个:

/plotme home 会带你去你1号地皮.

/plotme home:2 会带你去2号地皮.

/plotme home[:#] 去别人家/plotme home id 会带你去他的1号地皮.

/plotme home:2 id 会带你去他的2号地皮.

/plotme list 列出你的地皮

/plotme list 列出别人的地皮

/plotme info 列出你站在的地皮的信息(包括地皮id, 主人,地形, 日期, 结束 帮助).

/plotme comment 评论一下（留言）到当前的地皮

/plotme comments 查看各种留言

/plotme biome 查看目前地形

/plotme biome 改变当前地皮的地形

/plotme biomelist 列出可设置地形

/plotme tp 传送到某个地皮

/plotme id 查看当前地皮的id和坐标

/plotme clear 将地皮初始化到认领时候的样子（地形哦~慎用）

/plotme reset 还原地皮（慎用，解除占领的）

/plotme add 给某个玩家在你地皮建造的权限

/plotme remove 移除某个玩家在你地皮建造的权限

/plotme setowner 过户你的领地给玩家

 

商城系统

使用菜单或右键主城点券商城npc进入商城

商城内可以购买VIP 礼包 银子等物品
